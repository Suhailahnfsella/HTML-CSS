Hobi


Hobi merupakan kegiatan rekreasi yang dilakukan pada waktu luang untuk menenangkan pikiran seseorang. kata Hobi merupakan sebuah kata serapan dari Bahasa Inggris "Hoby".
Berikut beberapa hobi saya dan penjelasannya:


  • Menonton Film

    • Ketika tugas sekolah, pekerjaan rumah saya sudah selesai, biasanya saya menggunakan waktu luang saya untuk menonton film. film yang biasa saya lihat kebanyakan tentang komedi, karna saya merasa terhibur dan tertarik dengan film itu.

  • Bermain game

    • Selain menonton film, saya juga suka bermain game. game yang saya mainkan bukan game online tetapi offline seperti game asah otak memindahkan bola sesuai warnanya. karena menurut saya itu sebuah tantangan dan membuat banyak berpikir untuk bermain game itu
